
#📊 Marven Market Analysis Dashboard (Power BI)
📌 Project Title

Marven Market Analysis Dashboard using Power BI

🎯 Purpose

The purpose of this project is to analyze sales performance, revenue trends, and product contributions of a retail business using an interactive Power BI dashboard.
It helps in understanding business performance across regions, products, and time periods to support data-driven decision-making.

🛠️ Tech Stack
Power BI – Data visualization & dashboard creation
Microsoft Excel / CSV – Data preprocessing
DAX (Data Analysis Expressions) – Calculations and measures
Power Query – Data transformation

📂 Data Source
Dataset sourced from Kaggle
Contains retail data including:
Product details
Sales regions
Store information
Revenue data
Customer locations

🚀 Features / Highlights
🔍 Problem Statement

Businesses often struggle to:

Track revenue trends over time
Identify top-performing products
Understand regional sales performance
Make quick decisions due to scattered data

🎯 Goal of the Dashboard
Provide a centralized view of business performance
Identify top products and stores
Analyze monthly and yearly revenue trends
Understand regional sales distribution

📊 Visuals Used
Bar Charts → Product-wise & Store-wise revenue
Donut Chart → Revenue contribution by products
Line/Scatter Chart → Revenue trends by month & year
Map Visualization → Sales by country/region
Table → Detailed product revenue
Slicers → Region, Month, Year filters

💡 Business Insights
📈 Identified top-performing products contributing maximum revenue
🏪 Store-wise comparison shows which stores generate higher sales
🌍 Regional analysis highlights strong and weak markets
📅 Monthly trends reveal seasonal patterns in revenue
📊 Contribution analysis helps focus on high-value products
